# 演示视频脚本：Zsiga 开学季 AR 情绪盲盒

建议视频时长：60-90 秒。

## 镜头顺序

1. 展示泡泡玛特 Zsiga 官方竖版海报和二维码。

旁白：

```text
这是一个围绕泡泡玛特 Zsiga 在开学季传播节点制作的 WebAR 互动广告，用户无需安装 App，只需扫码进入网页。
```

2. 手机扫码或打开链接。

旁白：

```text
页面打开后，浏览器会请求摄像头权限。
```

3. 点击允许摄像头。

旁白：

```text
授权后，用户将手机摄像头对准泡泡玛特 Zsiga 官方竖版海报。
```

4. 对准官方海报，展示 AR 内容出现。

旁白：

```text
识别成功后，Zsiga 情绪盲盒会叠加在真实画面上，完成从线下海报到互动入口的第一层转化。
```

5. 点击“开盒”。

旁白：

```text
开盒层强化主题“生来自由，倔强生长”，让角色从平面海报中被唤醒。
```

6. 点击“刺刺”。

旁白：

```text
刺刺层不再只是说明，而是一个态度触发动作。用户点击“让我慢热 / 给我点空间 / 按我来”，就能点亮今天的边界宣言。
```

7. 点击“能量”。

旁白：

```text
用户在能量层获得一张属于自己的开学情绪卡，把开学季的不安和自我表达转成可分享的情绪结果。
```

8. 点击“抽盒”。

旁白：

```text
抽盒层会弹出本轮收盒结果，并出现“进入泡泡抽盒机”的品牌入口卡。这样评委能直接看到，这个作品后面接的是分享传播和品牌转化，而不是停在展示页。
```

9. 展示完整体验收尾。

旁白：

```text
当前基础正式版已经完成扫码进入、摄像头授权、官方海报识别、AR 叠加和点击交互，可继续部署 HTTPS 链接并录制最终提交视频。
```

## 拍摄注意

- 用另一台手机录制操作过程，尽量同时拍到海报和识别后的手机画面。
- 官方竖版海报放在光线充足、无遮挡的环境里。
- 录“刺刺”时，记得真的点一条态度签，让页面出现被触发后的文案变化。
- 每个按钮停留 2-3 秒，方便评委看清内容变化。
- 视频里要出现完整流程，不要只录最终 AR 效果。
